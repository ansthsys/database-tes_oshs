Cara menjalankan Aplikasi: 
- Download file .rar atau .zip lalu extract ke dalam folder htdocs (XAMPP) atau www (Laragon)
- Mengaktifkan SQLite dengan cara menghapus komen/tanda ';' pada baris extension=pdo_sqlite dan extension=sqlite3 di file php.ini
- Jalankan lokal server (XAMPP/Laragon)
- Buka browser dan ketikkan http://localhost/database-tes_oshs

Ketentuan: 
- Aplikasi yang dikerjakan sudah sesuai dengan ketentuan yang diberikan perusahaan
- Aplikasi menggunakan Bootstrap untuk styling
- Menggunakan metode MVC sederhana buatan sendiri
- Menggunakan PHP Native tanpa framework
- Memiliki fitur Input, Edit, dan Hapus
- Hanya menggunakan 1 halaman untuk melakukan fitur CRUD
